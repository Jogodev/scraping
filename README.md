# Scraping project in python
## Start
### First clone the project:
````bash
$ git clone https://github.com/Jogodev/scraping.git
$ cd scraping
````
### Install all the requirements:
````bash
$ pip install -r requirements.txt
````
### Make some data:
````bash
python scrap.py
````
